Fundamentals of Linear Regression

Linear Regression is a powerful machine learning algorithm used for predicting a continuous target variable based on one or more input features.

1. Simple Linear Regression

It models the relationship between a single input feature (x) and the target variable (y), expressed as: y = mx + b.

2. Multiple Linear Regression

This extends the concept to multiple input features (x₁, x₂, ..., xn) and is expressed as: y = θ₀ + θ₁x₁ + θ₂x₂ + ... + θₙxₙ.

3. Gradient Descent

An optimization algorithm used in training linear regression models. It iteratively adjusts the model's coefficients to minimize the cost or loss function. The update rule is: θᵢ = θᵢ - α * ∂(Cost Function) / ∂(θᵢ).
